# Print Filament Swapper

A static React + Tailwind SPA for planning multi-color 3D printer filament swaps.

This is a hobby project for hobby printers, not meant to be a serious application for print farms and the like.

## Run

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```

## Deploy

Deploy the `/dist` folder as a static website.
